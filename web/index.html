The Fizz Buzz Generator

This is the fizz buzz generator.

A lot of programming interviews require their candidates to program Fizz Buzz.

Here are the rules of Fizz-Buzz.


If the number is divisible by 3, then the user outputs Fizz.

If the number is divisible by 5, then the user outputs Buzz.

If the number is divisible by 3 and 5 (or 15 to be easier).

Enter the number you would like to stop at.
